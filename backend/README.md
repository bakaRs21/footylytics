## SETUP

### Create virtual environment

**Press**
    : crt + shift + p

**Type**
    -> Python: Create Environment -> venv -> Select Interpreter -> Select Folder -> Select Base Interpreter -> Wait for creation -> Select Created Interpreter

#### Activate virtual environment:
- Windows:
```bash
.\venv\Scripts\activate
```
- MacOS/Linux:
```bash
source ./venv/bin/activate
```

### Install dependencies

```bash
pip install -r requirements.txt
```
